nanotune.tuningstages.gatecharacterization_tasks.get_new_gatecharacterization_range(current_valid_ranges: Sequence[Sequence[float]], safety_voltage_ranges: Sequence[Sequence[float]], range_update_directives: List[str]) → Sequence[Sequence[float]][source]ÔÉÅ

Determines new voltage range for a subsequent tuning stage iteration. It extends the current range to the relevant safety range, i.e. if the directive required a voltage to be swept more negative, then the lower bound is set to the lower safety value. No intermediate tries, just being efficient.

Parameters

  • current_valid_ranges ‚Äì Current voltage ranges.

  • safety_voltage_ranges ‚Äì List of safety ranges.

  • range_update_directives ‚Äì List of range update directives.

Returns

list ‚Äì List with a tuple with the new voltage range.

Checks the average strength of measured signal over a given voltage interval is below the noise floor.

If this is the case, the boolean returned indicates that the measurement can be stopped.

Parameters

  • last_measurement_strength ‚Äì Last measurement output.

  • normalization_constant ‚Äì Constant to normalize last_measurement_strength.

  • recent_measurement_strengths ‚Äì List of most recent signal strengths.

  • voltage_precision ‚Äì Voltage precision of the measurement, i.e. voltage difference between setpoints.

  • noise_level ‚Äì Relative noise in percent compared to normalized signal.

  • voltage_interval_to_track ‚Äì Voltage interval over which the average should be taken. Voltage ranges less than voltage_interval_to_track will be ignored, i.e. will always return False.

Returns

  • bool ‚Äì Whether or not a measurement can be stopped.

  • list ‚Äì List of measurements strenghts/outputs over the last voltage_interval_to_track.
